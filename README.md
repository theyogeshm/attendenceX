# Attendance Tracker

A dynamic attendance tracking application with a clean dashboard, timetable integration, and analytics.

## Features
- Dynamic subject cards based on timetable.
- Mark attendance (Present, Absent, Miss, Leave).
- Clear last attendance entry.
- Analytics and stats (Attendance percentage, can miss lecture calculator).
- Profile customization with bio and social links.
- Dark mode support.

## Project Structure
- `index.html`: Main frontend (Vanilla JS version).
- `jss.js`: Frontend logic and API integration.
- `style.css`: Main stylesheet.
- `backend/`: Node.js/Express server using `lowdb` for storage.

## Deployment Note
The frontend can be hosted on **GitHub Pages**, but it requires the **Backend API** to be hosted on a platform like **Render** or **Railway** to function fully.

Currently, the frontend is configured to point to `localhost:5000`.
